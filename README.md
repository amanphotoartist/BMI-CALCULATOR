# BMI Calculator

A simple BMI (Body Mass Index) Calculator application built with Tkinter for the GUI and SQLite for data storage. The application allows users to input their weight and height, calculate their BMI, categorize the result, and store the data for future reference. Users can also view their BMI history plotted over time.

## Features

- **BMI Calculation:** Input weight and height to calculate BMI and display the category.
- **Data Storage:** Store the calculated BMI, weight, height, and category in an SQLite database.
- **View History:** View the BMI history plotted over time using Matplotlib.

## Prerequisites

- Python 3.x
- Tkinter (usually included with Python installations)
- SQLite (usually included with Python installations)
- Matplotlib (`pip install matplotlib`)
